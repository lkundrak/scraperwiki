ScraperWiki supports a number of 3rd party Perl libraries that we recommend for screen scraping, data analysis and data visualisation.

If you would like us to add a library that isn't listed here, please get in touch.

Scraping and refining

HTML::Parser
A versatile and robust callback-driven HTML parser. docs
HTML::TreeBuilder
Parser that produces a syntax tree using HTML::Parser. docs
XML::Simple
Transforms XML document into structure of hashes and arrays based on heuristics. Particularly useful for quick and dirty XML file reading. docs
XML::Smart
Parses an XML file, returning a structure of objects tied to dictionaries and hashes at the same time, letting you choose a prefered way to access the tree. docs
XML::LibXML
Low-level interface to libxml library, implementing XPath and more. docs
LWP::Simple
Easy way to fetch a HTTP resource. docs
WWW::Mechanize
Programmatic web browser, that navigates links, submits forms, keeps cookies and deals with authentication. docs
Spreadsheet::ParseExcel
Read XLS Excel files. docs
Spreadsheet::XLSX
Read XLSX Excel files (post-2007). docs
Text::CSV_XS
A fast way to access CSV data. docs
